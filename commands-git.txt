Pour cloner le repository, jai taper sur le terminal :

$git clone https://github.com/aicha-camara/git-begins.git

le dossier ce cree sur le pc je deplace la documentation a l’interieur puis :

$ git add . 
$ git commit -m "git begin"
$ git remote add origin https://github.com/aicha-camara/git-begins.git
$ git branch -M main
$ git push -u origin main

Il faudra rentrer ces identifiant et il apparaitra dans GIT HUB

Les  3 commandes de push sont: 
 - git push origin [branche]
 - git push -u origin [branche]
 - git push --all

Pour la commande il faut :
$ git pull
$ git fetch
$ git merge
